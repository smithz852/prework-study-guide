HTML

  • Hyper text markup language
  • Common elements: Text elements, Image elements, Header elements, List elements
  • Dividing and descriptive elements: Meta-related, Content division, Semantic
  • Metadata - info about webpage that the browser uses to relay webpage to users correctly
  • Metadata is located in the head elements
  • Metadata tags inside element:
    1. title - defines title for webpage. Important for SEO"
    2. style - defines CSS styles for page
    3. base - defines base URL
    4. link - connects external resources to webpage
    5. meta - defines other metadata. More...

  • body element - content visible to users. Semantic HTML elements
  • Body element content is contained between an opening tag and a closing tag
    1. Ex: head ... /head
  • Body Element Types:
    1. h - h1, h2, etc. are headings
    2. p - paragraph or block of text
    3. ul - unordered list
    4. ol - ordered list
    5. li - list items (bullets)
    6. img - images. Use src attribute for specifying source of image
    7. br - break, empty space
    8. a - anchor, used for hyperlinks. Use href attribute to insert hyperlink
    9. alt - text string describing appearance and functionality of image
  • section elements - added to body to create sections

CSS

  • Cascading Style Sheets - how info is presented to users, styles
  • CSS styling types:
    1. Inline css - style directly into HTML file, applies to a specific HTML tag. Good for quick permanent changes but not flexible
    2. Internal css - applied in head with text style to define style rules for webpage
    3. External css - most common, allows for all CSS rules to be in separate files
  • Margin - for margins outside of an element,space
  • Padding - space around content inside an element

JavaScript

  • To make the webpage more interactive
  • Variable - named container to store code in
  • Var - declares variable, allows us to name it, = then allows us to assign specific data
  • Control flow - order that the computer executes script
  • Array - single variable that holds a group of data
  • for Loop - uses predictable pattern to perform a task based on items in array
  • Loops contain:
    1. Starting point, i.e. var x=0
    2. Condition, i.e. x < topics.length
    3. Final statement, allows array to iterate in a predictable pattern, i.e x++, it increases by 1 each time
  • Function - instruction telling the computer how to perform a task
